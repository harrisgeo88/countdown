## Usage

Specify duration in the following format `1h2m3s`.

```bash
yarn countdown 25s
```

## Key binding
* space: To pause / resume the countdown.
* Esc or Ctrl+C: To stop the countdown without running next command.

## Install

```
yarn clone https://github.com/harrisgeo88/countdown
cd countdown
yarn install
```
